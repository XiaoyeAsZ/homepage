## Abstract

SpMMPlu is a compiler plug-in for sparse matrix multiplication based on a flexible Sparse IR. The IR represents different sparsity patterns by decomposing computation into compact dense meta-operations, enabling compiler passes that generate faster sparse kernels for inference workloads.
