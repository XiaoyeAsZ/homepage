## Abstract

SALO is a spatial accelerator for hybrid sparse attention in long-sequence Transformer workloads. It maps diverse sparse attention patterns onto hardware through a data scheduler and specialized compute architecture, reducing the cost of attention compared with general CPU and GPU execution.
