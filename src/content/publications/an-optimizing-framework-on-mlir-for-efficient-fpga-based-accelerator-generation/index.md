## Abstract

This work presents POM, an MLIR-based optimization framework for FPGA accelerator generation. POM uses multi-level intermediate representations, polyhedral loop transformations, a scheduling-oriented DSL, and automated design-space exploration to generate high-performance FPGA accelerators with less manual HLS tuning effort.
